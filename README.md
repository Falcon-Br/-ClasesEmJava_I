# -ClasesEmJava_I
Repositório destinado aos exercícios de POO da instuição de Ensino Fucapi.

## Estrutura das Pastas individuais

O workspace contém duas pastas por padrão, onde:

- `src`: Caminho que contém todo o conteúdo dos programas.
- `bin`, `.vscode`: Ignorar esses diretórios, pois são para configuração da IDE e do programa.


## Estrutura dao repositório

Todas as pastas estão organizadas dentro desse mesmo repositório, cada pasta é referente a um exercício.
